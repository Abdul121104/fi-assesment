## Prerequisites
- Node.js (v16+ recommended)

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Abdul121104/fi-assesment
   cd /frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend dev server:
   ```bash
   npm run dev
   ```
   The app will run on `http://localhost:5173` by default.
